# Overview

## Description

NeurIPS Open Polymer Prediction 2025 focuses on predicting specific properties of polymers based on their structural data. Participants are provided with datasets containing polymer structures and are tasked with developing models to accurately predict their properties.

## Evaluation

### Metric

Submissions are evaluated using Mean Absolute Error (MAE). MAE measures the average magnitude of errors between predicted and actual values, providing a straightforward assessment of model accuracy.

### Submission Format

The submission format is a csv file with predicted property values for each test sample.
