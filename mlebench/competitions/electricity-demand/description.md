# Overview

## Description

Electricity Demand is a time series forecasting competition focused on predicting electricity demand. The competition involves forecasting future electricity consumption based on historical data.

## Evaluation

### Metric

Submissions are evaluated using accuracy, measuring the proportion of correct predictions for electricity demand class labels (UP/DOWN).

### Submission Format

The submission format is a csv file with predicted electricity demand values for each test sample.

## Dataset

The dataset contains historical electricity demand data with temporal features that can be used to forecast future demand.
